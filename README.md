# Telegram-bot-OpenLibBot-
# 📚 OpenLibBot — Telegram бот для поиска книг

# 📖 О проекте

**OpenLibBot** — это Telegram-бот для поиска книг по названию или автору. Бот использует публичное API [Open Library](https://openlibrary.org) для получения информации о книгах и предоставляет удобный интерфейс с инлайн-кнопками.

### ✨ Возможности

- 🔍 **Поиск книг** по названию и автору
- 📖 **Пагинация** результатов (по 5 книг на странице)
- ⭐ **Быстрые запросы** (популярные книги и классические авторы)
- 🎲 **Случайная книга** для вдохновения
- 🧭 **Удобная навигация** с инлайн-кнопками
- ❓ **Встроенная справка** и информация о боте
- ⚡ **Асинхронная работа** для быстрых ответов
- 🛡️ **Обработка ошибок** и таймаутов

## 🛠️ Используемые технологии

| Технология | Версия | Назначение |
|------------|--------|------------|
| **Python** | 3.10+ | Основной язык разработки |
| **python-telegram-bot** | 20.7 | Взаимодействие с Telegram Bot API |
| **aiohttp** | 3.9.1 | Асинхронные HTTP-запросы к Open Library API |
| **python-dotenv** | 1.0.0 | Загрузка переменных окружения |
| **asyncio** | встроенная | Асинхронное выполнение |
| **Open Library API** | — | Внешний источник данных о книгах |

## 📋 Требования к установке

- Python 3.10 или выше
- pip (менеджер пакетов Python)
- Telegram аккаунт
- Токен бота от [@BotFather](https://t.me/BotFather)

## 🚀 Установка и запуск

### Шаг 1: Клонирование репозитория

```bash
git clone https://github.com/ваш_аккаунт/openlibbot.git
cd openlibbot

### Шаг 2: Создание виртуального окружения
Windows:
python -m venv venv
venv\Scripts\activate

macOS/Linux:
python3 -m venv venv
source venv/bin/activate

### Шаг 3: Установка зависимостей
pip install -r requirements.txt

### Шаг 4: Настройка токена
Создайте файл .env в корне проекта и положите в него:
TELEGRAM_BOT_TOKEN=ваш_токен_от_botfather

### Шаг 5: Запуск бота
python TelegramBot.py

При успешном запуске будет выведен текст:
✅ Бот запущен!
